# FakeShutDown

一个轻量的 Windows 小工具：在所有显示器上创建“关机后那种全黑全屏”的覆盖效果，直到按下 **Esc** 退出。

> ⚠️ 注意：FakeShutDown **不会**真的关机、休眠、注销或锁屏；它只是用全屏黑色置顶窗口把屏幕内容“盖住”。  
> 这不是安全边界（security boundary），也不等同于 Windows 锁屏。请在合规、被允许的场景下使用。

---

## 使用场景（包括但不限于）

- 临时离开工位，想快速“黑屏遮挡”屏幕内容（防窥/演示/会议切换）。
- 下班的时候不想关电脑，但又不希望屏幕继续亮着或显示内容（例如在办公室里）。
- 展示/展厅/投屏环境：需要一键进入“黑场”效果。

---

## 特性

- ✅ 多显示器支持：每个显示器一张全黑全屏覆盖层
- ✅ 置顶、无边框、无任务栏图标（更像“突然黑了”）
- ✅ 隐藏鼠标光标
- ✅ **Esc** 一键退出
- ✅ 纯 Win32 API，无额外依赖

---

## 下载与运行

### 直接运行
1. 下载或自行编译得到 `FakeShutDown.exe`
2. 双击运行
3. 退出：按下 **Esc**

建议：给它做个桌面快捷方式，甚至绑到鼠标侧键/键盘宏上，体验更顺手。

---

## 从源码编译

### 环境要求
